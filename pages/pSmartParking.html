Contexto del proyecto

¿Qué es SmartParking?

SmartParking nació en el Hackatec, un hackathon de 24 horas organizado por el ITCH2 y patrocinado por HikVision. El reto era crear una solución tecnológica usando el ecosistema de cámaras y software de la empresa.

La idea fue construir un sistema que permitiera a cualquier persona ver desde su dispositivo qué cajones del estacionamiento están libres y cuáles ocupados, sin necesidad de recorrerlo físicamente.

Mi contribución

¿Qué hice yo?

Dentro del equipo de 5 personas me encargué de tres frentes: adaptar la aplicación de C# que HikVision proporcionó para que enviara los datos de detección a mi sitio web, construir la base de la página web que recibía y visualizaba esos datos, y dar el pitch frente al jurado.

🔌
Integración con HikVision SDK
Adapté la aplicación de C# proporcionada por HikVision para que, en lugar de solo mostrar datos localmente, los enviara en tiempo real al sitio web vía WebSocket.
🌐
Desarrollo del sitio web
Construí la base de la página en HTML, CSS y JavaScript con un listener de WebSocket que recibía el estado de cada cajón y actualizaba la visualización al instante.
🎤
Pitch ante el jurado
Presenté el proyecto y sus fundamentos técnicos frente al panel de jueces del hackathon, comunicando tanto la solución como su potencial de escalabilidad.
🔗
Infraestructura de red
La cámara se conectó mediante switch y router para garantizar la comunicación estable entre el hardware de HikVision, la app de C# y el servidor web local.
Arquitectura

¿Cómo funciona?

El flujo de datos va de la cámara al sitio web en tres pasos:

📷
1 — Detección por cámara
La cámara HikVision analiza el área del estacionamiento y detecta si un cajón cambia de estado (libre → ocupado o viceversa).
⚙️
2 — Procesamiento en C#
La app de C# (adaptada del SDK de HikVision) recibe el evento de la cámara y lo retransmite al sitio web mediante un servidor WebSocket.
🖥️
3 — Visualización web
El sitio web escucha el WebSocket y actualiza en tiempo real la representación visual de cada cajón, mostrándolo en verde (libre) o rojo (ocupado).
